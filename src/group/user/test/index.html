曾小洁老师

专业舞蹈老师

舞风优雅细腻，教学严谨清晰； 受启蒙及师从国内导师秀林&赵学伟老师，基本功功底扎实。 曾留任秀林东方舞艺术培训中心担当执教老师， 曾参加俄罗斯世界著名导师Aida Bogomolova和Alla Vatc的集训课程； 曾参加2017深圳东方舞职业大赛获得优秀奖。 2017惠州东方舞职业大赛决赛十佳舞者。 并多次参加大型演出活动及担当舞团领舞工作。

唐微微老师

资深爵士舞老师

资深爵士舞导师，自幼学习爵士舞，热爱舞蹈的她，专注日韩爵士舞，欧美爵士舞，TB秀、领舞等现代舞等多种不同风格的舞蹈领域，从教多年，不断创新，有着丰富的教学经验和现场激情表演经验，深受爵士舞学员们的喜爱。

王九莲老师

认证瑜伽导师

人力资源和社会保证部认证导师瑜伽师，毕业于济南禅悦瑜伽学校，系统的学习了包括哈他、阴瑜伽、产后修复、理疗、商卡瑜伽的课程；具有多年丰富的瑜伽导师教学经验，很大程度的发掘每位学员的优势。使每位同学上岗都能具备自己的教学风格；教学风格轻松而愉悦，同时具责任的的教学态度，深受同学们爱戴。

于蔷薇老师
